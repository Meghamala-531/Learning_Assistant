# Phase 8: Project Demonstration 🎬

This phase documents walkthrough details and instructions for demonstration of the EduGenie Assistant.

---

## 1. Interface Overview
EduGenie provides a sleek, modern, single-page web dashboard using CSS glassmorphism, glowing gradients, responsive sidebar navigation, and HSL palettes:

- **💬 Question Answering**: Submits questions and displays formatted answers alongside details of the model used (Groq vs Local).
- **💡 Concept Explainer**: Displays explanations using real-world analogies.
- **📝 Quiz Generator**: Renders MCQs with a clean, toggleable "Show Answer Key" drawer.
- **🔍 Text Summarizer**: Compiles summaries into readable bulleted outlines.
- **🗺️ Learning Paths**: Creates stage-by-stage learning roadmaps.

---

## 2. Interactive Demonstrations

### Scenario A: Question Answering
- **Input**: *"Which is the largest ocean?"*
- **Action**: Click "Ask EduGenie".
- **Output**: Returns "The Pacific Ocean is the largest..." formatted in markdown.

### Scenario B: Interactive Quiz
- **Input**: *"Pythagorean Theorem"*
- **Action**: Click "Generate Quiz".
- **Output**: Renders the multiple choice questions, and shows a "Show Answer Key" button which expands a drawer displaying the answers.
