# Phase 1: Brainstorming & Ideation 🧞‍♂️

## 1. Project Conceptualization
EduGenie was conceived to address a key educational gap: providing high-quality, personalized study guidance that remains accessible and lightweight. 

## 2. Target Audience
* **Self-Learners & Professionals**: Seeking structured roadmaps to transition into new skills (e.g., learning SQL or Cyber Security).
* **Students**: Needing immediate, intuitive explanations of complex academic concepts (e.g., photosythesis) or exam preparation.
* **Educators**: Needing an assistant to generate self-assessment quizzes or summarize lengthy academic materials.

## 3. Ideated Features
1. **Intelligent Question Answering (QA)**: Rich educational answers with context.
2. **Concept Explainer**: Simplifies terms using relatable real-world analogies.
3. **AI Quiz Generator**: Creates multiple-choice questions with answer key separation.
4. **Text Summarizer**: Highlights takeaways from pasted pages or study notes.
5. **Learning Paths**: Structured roadmaps divided into stages (Beginner, Intermediate, Advanced).
